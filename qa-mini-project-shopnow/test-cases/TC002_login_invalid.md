# TC002 — Login com senha inválida

## Pré-condição
- Usuário cadastrado  
- Usuário deslogado  

---

## Passos

1. Acessar a tela de Login  
2. Inserir email válido  
3. Inserir senha inválida  
4. Clicar em **Entrar**

---

## Resultado esperado

O sistema deve impedir o login e exibir:

❌ “Senha inválida”
